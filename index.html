Javascript(JS)

Javascript is a programming language used to make web pages interactive.

Operators

An operator is a special character (or characters) that indicates an action to be performed.

Variables

Variables are containers that hold information; this can be mathematical numbers, strings(letters/words), Booleans (True or False), and others.

Variable Naming Conventions

Variables are written in lower camelcase. (thisIsAnExample)

Functions

Functions perform actions in the browser. Functions are an incredibly important part of programming, and you'll be using them constantly.

Methods

You can think of a method as an action that a number can perform or have taken on it.

Arguments

Arguments provide a bit more information to methods to help them know what they're supposed to do.

Parameters

A parameter is a variable, function, or method that we "hand to" a function so it can perform its operations. For example, a function to slice produce does no good if we don't also hand it something to be sliced and a knife of some sort.

Return

The entries after Return are what a function hands back when it has completed (if anything). In the sliced-produce example: it would be the equivalent of "Return: sliced produce"

Chaining Methods

Chaining methods is a handy way of compacting code into one place; we can chain methods to perform many operations "all at once but in sequence", rather than having to perform them piece-meal.

Strings

Strings are a type of data structure. Strings are a series of characters, "on a string".

Booleans

A Boolean is a data type and can only hold 2 possible values true or false

Undefined

Undefined is data type has only one possible value - undefined. It simply represents that nothing has been returned from the function, or that a variable hasn't been assigned a value.

NaN

Nan is a number data type and stands for: “not a number”

Escape

a "\" used in a string to denote a non-coding character. For example "\)" would mean "do not treat this closing bracket as a piece of code".

Alerts

An alert() is a function that can take an argument. The alert() function will pop up a dialog box with the string that you pass in as an arugement

Comments

You can insert comments into your text editor by using a double back-slash (like so: //) to insert code or comments after the double back slash that will not be visible to UI

JQuery

A commonly-used Javascript Library.

Attributes

Attributes are characteristic-holding variables assignable to tags.
